# PrecisionStartList Document Type Setup

## Document Type Creation (Umbraco Backoffice)

**Location:** Settings → Document Types

### 1. Create Document Type
- **Name:** Precision Start List
- **Alias:** `precisionStartList`
- **Icon:** icon-list (or similar list icon)
- **Allow as Root:** No
- **Allow at Root:** No

### 2. Template
- **Create Template:** Yes (will auto-create PrecisionStartList.cshtml)
- **Master Template:** master

### 3. Properties Tab: "Start List Details"

| Property Name | Alias | Data Type | Description |
|---------------|-------|-----------|-------------|
| Competition ID | `competitionId` | Numeric | Links to competition content |
| Team Format | `teamFormat` | Dropdown | Mixed teams vs separated classes |
| Start List Content | `startListContent` | Rich Text Editor | Generated HTML content |
| Configuration Data | `configurationData` | Textarea | JSON configuration data |
| Generated Date | `generatedDate` | Date Picker | When list was generated |
| Generated By | `generatedBy` | Textbox | Who generated the list |
| Notes | `notes` | Textarea | Optional notes |
| Is Active | `isActive` | True/False | Whether list is active |

### 4. Team Format Dropdown Values
Add these values to the Team Format dropdown:
- **Mixade Skjutlag** - Mixade vapengrupper per skjutlag
- **En vapengrupp per Skjutlag** - En vapengrupp per skjutlag

### 5. Allowed Child Content Types
None (this is a leaf content type)

### 6. Allowed Parent Content Types
- Competition
- Competitions Hub (if exists)

### 7. Permissions
Configure appropriate permissions for:
- Administrators: Full access
- Club Admins: Create/Edit for their competitions
- Regular Users: View only

## After Creation

1. **Save** the document type
2. **Verify** the template was created at `Views/PrecisionStartList.cshtml`
3. **Test** by creating a test start list content item

## Usage

Once created, start lists can be generated via:
- **Admin Interface:** Competition Management → Start Lists tab
- **Direct API:** `/umbraco/surface/StartList/GenerateStartList`
- **Manual Creation:** Content → Create → Precision Start List

The generated content will have URLs like:
`/precision-start-list-12345`
